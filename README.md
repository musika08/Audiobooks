Audiobooks is a simple desktop app that turns your text into narrated audiobooks.
It comes with easy settings for book metadata, chapter intros/outros, audio mastering, and export options.

🚀 Features

Add cover image, title, author, and album

Add intro and outro for chapters

Apply basic audio mastering

Choose bitrate and output format

Works with FFmpeg (auto-installed if missing)

📦 Installation
